# ASCII 可视化技能

在纯文本环境中，用**可打印字符**构建可视化结构，让信息更直观。

---

## 1. 字符选择

### 常用字符集

```
边框字符:
┌ ┐ └ ┘ │ ─ ├ ┤ ┬ ┴ ┼
╔ ╗ ╚ ╝ ║ ═ ╠ ╣ ╦ ╩ ╬

装饰字符:
◆ ◇ ■ □ ○ ● ▲ △ ▼▽
★ ☆ ♦ ♢ ♣ ♠ ♥

箭头:
← → ↑ ↓ ↔ ↕ ⇒ ⇐ ⇑ ⇓
>> => -> <- <=

连接线:
─ │ ━ ┃ └ ┘ ├ ┤ ┬ ┴

点线:
· · · · ·  ┅ ┅ ┅ ┅ ┅
```

### 用途分类

| 类型 | 字符 | 用途 |
|------|------|------|
| 粗边框 | ┌─┐ │  | 专业外观的方框 |
| 双线 | ╔═╗ ║ | 强调/标题区域 |
| 虚线 | ┅ ┆ | 分隔/过渡 |
| 点 | · • ● | 列表/要点 |

## 2. 表格

### Markdown 表格

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |
```

### 纯文本表格

```
┌───────┬───────┬───────┐
│ 列1   │ 列2   │ 列3   │
├───────┼───────┼───────┤
│ A1    │ B1    │ C1    │
├───────┼───────┼───────┤
│ A2    │ B2    │ C2    │
└───────┴───────┴───────┘
```

### 双线风格

```
╔═══════╦═══════╦═══════╗
║ 列1   ║ 列2   ║ 列3   ║
╠═══════╬═══════╬═══════╣
║ A1    ║ B1    ║ C1    ║
╠═══════╬═══════╬═══════╣
║ A2    ║ B2    ║ C2    ║
╚═══════╩═══════╩═══════╝
```

## 3. 流程图

### 线性流程

```
[开始] ──► [步骤1] ──► [步骤2] ──► [步骤3] ──► [结束]
```

### 条件分支

```
              ┌─────────┐
              │ 判断条件 │
              └────┬────┘
                   │
         ┌─────────┴─────────┐
         │                   │
      Yes │                │ No
         ▼                   ▼
   ┌─────────┐        ┌─────────┐
   │  分支 A │        │  分支 B │
   └────┬────┘        └────┬────┘
        │                   │
        └─────────┬─────────┘
                  ▼
            [合并后继续]
```

### 编号步骤

```
① 读取输入
   │
   ▼
② 验证数据
   │
   ▼
③ 处理业务
   │
   ▼
④ 返回结果
```

## 4. 树形图

### 经典风格

```
根节点
├── 分支1
│   ├── 叶子1.1
│   └── 叶子1.2
├── 分支2
│   ├── 叶子2.1
│   └── 叶子2.2
└── 分支3
    └── 叶子3.1
```

### 符号说明

```
├──  → 带有同级兄弟的节点
└──  → 最后/唯一子节点
│    → 垂直连接线
    → 缩进（2 或 4 空格）
```

### 垂直树形

```
        根
        │
    ┌───┴───┐
    │       │
   子1     子2
    │       │
  ┌─┴─┐   ┌─┴─┐
  │   │   │   │
 叶1 叶2  叶3 叶4
```

## 5. 方框布局

### 单线边框

```
┌─────────────┐
│   内容      │
└─────────────┘
```

### 双线边框

```
╔═════════════╗
║   内容      ║
╚═════════════╝
```

### 带标题

```
┌─ 标题 ─────┐
│            │
│   内容     │
│            │
└────────────┘
```

### 双列布局

```
┌────────────────┐  ┌────────────────┐
│    左侧        │  │    右侧        │
│                │  │                │
└────────────────┘  └────────────────┘
```

### 嵌套布局

```
┌────────────────────────────────┐
│           外层                 │
│  ┌──────────┐   ┌──────────┐  │
│  │   内层1  │   │   内层2  │  │
│  └──────────┘   └──────────┘  │
│                                 │
└────────────────────────────────┘
```

## 7. 快速参考

```
常用字符:
方框: ┌ ┐ └ ┘ │ ─
双线: ╔ ╗ ╚ ╝ ║ ═
箭头: ← → ↑ ↓
装饰: ◆ ◇ • ★

布局原则:
• 使用等宽字体
• 保持一致的间距
• 先画骨架再填内容
• 从大到小构建

选择指南:
表格   → 对比、参数、规格
流程图 → 步骤、逻辑、决策
树形图 → 目录、层级、继承
方框   → 区域、组织、界面
```

       └────┘

       ┌─────────┐
       │  服务器 │

            │
