# ASCII 方框布局技巧

## 核心理念

方框布局是**组织信息**的核心方式，通过嵌套组合创建复杂结构。

## 1. 边框字符

### 单线边框

```
┌─────┐
│ 内容│
└─────┘

┌─────────────┐
│             │
│   内容      │
│             │
└─────────────┘
```

### 双线边框

```
╔═════╗
║ 内容║
╚═════╝

╔═════════════╗
║             ║
║   内容      ║
║             ║
╚═════════════╝
```

### 混合边框（单双混合）

```
╭─────╮
│ 内容│
╰─────╯

╭─────────────╮
│             │
│   内容      │
│             │
╰─────────────╯
```

### 虚线边框

```
┅┅┅┅┅
┅ 内容 ┅
┅┅┅┅┅
```

## 2. 基础布局模式

### 单列布局

```
┌────────────────────────────┐
│          标题              │
├────────────────────────────┤
│                            │
│         内容区域           │
│                            │
│                            │
└────────────────────────────┘
```

### 双列布局

```
┌────────────────┐  ┌────────────────┐
│    左侧        │  │    右侧        │
│                │  │                │
└────────────────┘  └────────────────┘
```

### 三列布局

```
┌────────┐  ┌────────┐  ┌────────┐
│  左侧  │  │  中间  │  │  右侧  │
│        │  │        │  │        │
└────────┘  └────────┘  └────────┘
```

## 3. 嵌套布局

### 简单嵌套

```
┌────────────────────────────────┐
│           外层                 │
│  ┌──────────┐   ┌──────────┐  │
│  │   内层1  │   │   内层2  │  │
│  └──────────┘   └──────────┘  │
│                                 │
└────────────────────────────────┘
```

### 多层嵌套

```
┌─────────────────────────────────────────┐
│  外层容器                              │
│  ┌───────────────────────────────────┐  │
│  │  第一层嵌套                       │  │
│  │  ┌─────────┐  ┌─────────┐       │  │
│  │  │ 第二层  │  │ 第二层  │       │  │
│  │  │ 嵌套A  │  │ 嵌套B   │       │  │
│  │  └─────────┘  └─────────┘       │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## 8. 快速参考

```
边框选择:
单线 ┌┐└┘│─  →  通用
双线 ╔╗╚╝║═  →  强调
圆角 ╭╮╰╯   →  友好
虚线 ┅        →  分隔

布局模式:
单列   → 内容流
双列   → 对比/并列
三列   → 导航+内容+侧边
栅格   → 卡片组

技巧:
先外后内  → 从大到小构建
先框后填  → 先画骨架再填内容
保持对齐 → 始终使用等宽间距
```
