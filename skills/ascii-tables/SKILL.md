# ASCII 表格技巧

## 核心理念

表格是**结构化对比**的最佳方式，让信息一目了然。

## 1. Markdown 表格（推荐）

### 基本结构

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |
```

渲染效果：

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |

### 对齐方式

```
左对齐 (默认):    |:--|--|
居中:            |:--:|--:|
右对齐:          |--:|--:|

示例:
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容   | 内容 |   内容 |
```

### 复杂示例

```
| 功能 | 免费版 | 专业版 | 企业版 |
|:-----|:-----:|:------:|-------:|
| 用户数 | 5 人 | 20 人 | 无限 |
| 存储空间 | 1 GB | 100 GB | 1 TB |
| 支持 | 社区 | 优先级 | 专属 |
| 价格 | ¥0/月 | ¥99/月 | 定制 |
```

| 功能 | 免费版 | 专业版 | 企业版 |
|:-----|:-----:|:------:|-------:|

## 2. 纯文本表格（等宽字体）

### 简单边框表格

```
┌───────┬───────┬───────┐
│ 列1   │ 列2   │ 列3   │
├───────┼───────┼───────┤
│ A1    │ B1    │ C1    │
├───────┼───────┼───────┤
│ A2    │ B2    │ C2    │
└───────┴───────┴───────┘
```

### 双线风格表格

```
╔═══════╦═══════╦═══════╗
║ 列1   ║ 列2   ║ 列3   ║
╠═══════╬═══════╬═══════╣
║ A1    ║ B1    ║ C1    ║
╠═══════╬═══════╬═══════╣
║ A2    ║ B2    ║ C2    ║
╚═══════╩═══════╩═══════╝
```

### 紧凑风格（无竖线）

```
───────────────
列1    列2    列3
───────────────
A1     B1     C1
A2     B2     C2
───────────────
```

### 点分隔风格

```
用户名 · 年龄 · 城市
────────────────
张三  · 28  · 北京
李四  · 35  · 上海
王五  · 42  · 深圳
```

## 3. 对齐技巧

### 垂直对齐（列内）

```
左对齐（默认）:
│ 短    │
│ 长文本│
│ 更长  │

右对齐:
│    短 │
│ 长文本│
│ 更长  │

居中:
│  短   │
│ 长文本 │
│ 更长  │
```

### 数字右对齐（推荐）

```
┌──────────┬──────────┐
│ 项目     │    金额  │
├──────────┼──────────┤
│ 收入     │  50,000  │
│ 支出     │  23,500  │
│ 利润     │  26,500  │
└──────────┴──────────┘
```

### 混合对齐

```
┌────────┬────────┬──────┐
│ 左对齐 │  居中  │ 右对齐│
├────────┼────────┼──────┤
│ 文本   │  数字  │  100 │
│ 内容   │  200   │   50 │
└────────┴────────┴──────┘
```

## 4. 高级表格

### 带合并单元格的模拟

```
┌────────────────────────────┐
│         标题行             │
├────────────┬───────────────┤
│   合并     │     列2       │
│   单元格   ├───────────────┤
│            │    列3        │
└────────────┴───────────────┘
```

### 多行单元格

```
┌────────────────────────┐
│ 描述                   │
│ 这是多行内容            │
│ 可以包含多行文字        │
├────────┬───────────────┤
│ 列1    │ 列2           │
│        │ 继续...       │
└────────┴───────────────┘
```

### 嵌套表格

```
┌─────────────────────────────────┐
│  外层表格                        │
│  ┌─────────┐ ┌─────────┐        │
│  │ 嵌套1   │ │ 嵌套2   │        │
│  └─────────┘ └─────────┘        │
└─────────────────────────────────┘
```

## 6. 选择指南

| 场景 | 推荐方式 |
|------|----------|
| 代码注释/纯文本 | 纯文本表格（┌ ┐ 风格） |
| Markdown 文件 | Markdown 表格 |
| 文档/报告 | 双线风格（╔ ╗ 风格） |
| 简单对比 | 点分隔风格 |
| 快速原型 | 紧凑无边框 |
