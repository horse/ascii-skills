# ASCII 可视化基础

## 核心理念

在纯文本环境中，用**可打印字符**构建可视化结构，让信息更直观：

- **表格** - 对比数据
- **流程图** - 展示步骤
- **树形图** - 层级关系
- **方框图** - 组织布局

## 1. 字符选择

### 常用字符集

```
边框字符:
┌ ┐ └ ┘ │ ─ ├ ┤ ┬ ┴ ┼
╔ ╗ ╚ ╝ ║ ═ ╠ ╣ ╦ ╩ ╬

装饰字符:
◆ ◇ ■ □ ○ ● ▲ △ ▼▽
★ ☆ ♦ ♢ ♣ ♠ ♥

箭头:
← → ↑ ↓ ↔ ↕ ⇒ ⇐ ⇑ ⇓
>> => -> <- <=

连接线:
─ │ ━ ┃ └ ┘ ├ ┤ ┬ ┴

点线:
· · · · ·  ┅ ┅ ┅ ┅ ┅
```

### 用途分类

| 类型 | 字符 | 用途 |
|------|------|------|
| 粗边框 | ┌─┐ │  | 专业外观的方框 |
| 双线 | ╔═╗ ║ | 强调/标题区域 |
| 虚线 | ┅ ┆ | 分隔/过渡 |
| 点 | · • ● | 列表/要点 |
| 数学 | ≤ ≥ ± × ÷ √ ∞ | 数学表达 |

## 2. 间距原则

### 等宽字体假设

所有 ASCII 艺术应在**等宽字体**下查看（如 Monaco, Consolas, Courier）。

```
错误（不等宽）:     正确（等宽）:
┌──────┐          ┌──────┐
│ 标   │          │ 标   │
└──────┘          └──────┘
   不整齐              对齐
```

### 列间距

```
标准间距: 2 空格
紧凑间距: 1 空格
宽松间距: 3-4 空格

示例:
名称    年龄    城市        ← 标准 2 空格
─────────────────────
张三    28     北京
李四    35     上海
```

### 行间距

```
元素之间: 1 空行
分组之间: 2 空行
区块之间: 3 空行
```

## 3. 对齐原则

### 左对齐（默认）

```
优先级:
1. 安全
2. 性能
3. 可维护性
```

### 右对齐（数字/标签）

```
文件名          大小
──────────────────────
report.pdf      2.3 MB
data.csv       15.7 KB
script.py       4.1 KB
```

### 居中对齐（标题）

```
═══════════════════════
     使用说明
═══════════════════════
```

### 垂直对齐

```
┌─────────┐   ┌─────────┐
│  区块1  │   │  区块2  │
│  内容   │   │  内容   │
└─────────┘   └─────────┘
   ↗ 对齐       ↖ 对齐
```

## 5. 实践技巧

### 5.1 先画骨架

```python
# 第一步：确定结构
+------+  +------+
|      |  |      |
+------+  +------+

# 第二步：填充内容
+------+  +------+
| A    |  | B    |
+------+  +------+

# 第三步：调整细节
┌──────┐  ┌──────┐
│ A    │  │ B    │
└──────┘  └──────┘
```

### 5.2 使用占位符

```
┌────────┐
│  X    │  ← 用 X 占位，稍后替换
└────────┘
```

### 5.3 分步构建

```
步骤1: 画出外框
████████

步骤2: 加内部分隔
████
████

步骤3: 细化内容
┌────┐
│内容│
└────┘
```

## 6. 常见错误与修正

### 问题1: 字符不一致

```
错误: ┌────┐
      │ 内容│  ← 右括号不同
      └────┘

修正: ┌────┐
      │ 内容│
      └────┘
```

### 问题2: 宽度不匹配

```
错误: ┌──────┐
      │内容│← 宽度不一致
      └─────┘

修正: ┌──────┐
      │  内容  │
      └──────┘
```

### 问题3: 间距不均匀

```
错误: a    b  c ← 间距不均

修正: a    b    c ← 等间距
```
